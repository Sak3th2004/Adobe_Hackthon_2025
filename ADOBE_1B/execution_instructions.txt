# How to build and run the ADOBE_1B project

## Step 1: Build the Docker image
Open your terminal and navigate to the ADOBE_1B directory, then run:
docker build -t adobe-challenge .

## Step 2: Run the Docker container
Run the container by mounting your local pdf_docs folder into the container:

docker run --rm \
  -v /absolute/path/to/ADOBE_1B/pdf_docs:/app/pdf_docs \
  -v /absolute/path/to/ADOBE_1B/results:/app/results \
  adobe-challenge

## Step 3: Output Location
The final results will be saved in the `results/` folder under the filename format:
- `<original_pdf_name>_output.json`

Each JSON contains:
- Metadata
- Extracted Section
- Sub-section Analysis
